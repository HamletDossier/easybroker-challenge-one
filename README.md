# Instrucciones para ejecutar el proyecto

1. Crea un archivo llamado .env y coloca la siguiente variable de entorno:
    ```env
    API_KEY=YOUR_API_KEY
    ```
    
2. Ejecuta el siguiente comando:
    ```sh
    bundle install
    ```

